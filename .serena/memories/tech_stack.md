# Technical Stack

- **Framework**: Next.js 15.5.18 (App Router)
- **Styling**: Tailwind CSS, Vanilla CSS custom tokens (e.g. glass-card)
- **Language**: TypeScript
- **State Management**: React useState, useCallback, useRef
- **Backend / DB**: Next.js server-side local JSON file database (`data/jps_punches.json`) with auto-repair mechanism.
